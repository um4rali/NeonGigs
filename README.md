## ğŸš€ Getting Started

### Prerequisites
- Node.js
- MongoDB Atlas Account
- Razorpay Developer Keys

### Backend Setup
```bash
cd backend
npm install

# Create .env file
echo "MONGODB_URI=your_connection_string
JWT_SECRET=your_secret
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret" > .env

npm run dev
```

### Frontend Setup
```bash
cd ../src
npm install
npm start
```
